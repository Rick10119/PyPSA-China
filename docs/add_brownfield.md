# add_brownfield.py 功能说明

## 输入输出关系
### 输入文件:
- network: 当前时间点的基础网络 (.nc)
- network_p: 上一时期的已求解网络 (.nc)
- costs: 成本参数 (costs_{planning_horizons}.csv)
- profile_{tech}: 可再生能源出力曲线 (.nc)
- overrides: 网络属性覆盖

### 输出文件:
- network_name: 更新后的网络文件 (.nc)

## 主要功能
1. **电网容量处理**:
   - 设置上一期优化结果为最小值
   - `n.lines.s_nom_min = n_p.lines.s_nom_opt`

2. **设施处理**:
   - 移除寿命到期的设施
   - 固定仍在运行的设施容量
   - 处理CHP（热电联产）设施

3. **可再生能源潜力更新**:
   - 更新陆上风电、海上风电和光伏的开发潜力
   - 考虑已建容量的占用
   - 确保潜力不为负值

4. **特殊处理**:
   - 2025年的特殊处理
   - 煤电厂碳捕集改造设置
